조작 방법
  • 퀴즈 화면: 정답이라 생각하는 문을 "클릭"하면 캐릭터가 자동으로 이동합니다.
  • 가위바위보 방 (패널티): 컴퓨터를 가위바위보로 이겨서 탈출하세요.
보상

3문제를 연속으로 맞히면 아이템 뽑기 기회가 주어집니다. 3개의 카드 중 하나를 선택해 아이템을 획득하세요!

아이템 설명
  • 힌트: 퀴즈 화면에서 사용 시, 4지선다형 문제의 오답 1개를 제거해 줍니다. (OX 문제에는 사용 불가)
  • 미래보기: '가위바위보 방'에서 사용 시, 컴퓨터의 선택을 미리 볼 수 있습니다.
  • 무승부=승리: '가위바위보 방'에서 사용 시, 컴퓨터와 무승부가 나도 승리로 처리됩니다.
