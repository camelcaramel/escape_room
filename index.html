게임 방법 안내

목표: 20개의 재난 안전 퀴즈를 풀고 최단 시간 안에 탈출하세요!

조작 방법
  • 퀴즈 화면: 정답이라 생각하는 포털을 "클릭"하면 캐릭터가 자동으로 이동합니다.
  • 시간의 방 (패널티): 화면의 "방향키"를 이용해 상하좌우로 날아오는 장애물을 피하세요!
보상

3문제를 연속으로 맞히면 아이템 뽑기 기회가 주어집니다. 3개의 카드 중 하나를 선택해 아이템을 획득하세요!

아이템 설명
  • 힌트: 퀴즈 화면에서 사용 시, 4지선다형 문제의 오답 1개를 제거해 줍니다. (OX 문제에는 사용 불가)
  • 시간 정지: '시간의 방'에서 사용 시, 5초 동안 패널티 타이머와 모든 장애물의 움직임이 멈춥니다.
  • 보호막: '시간의 방'에서 사용 시, 1회에 한해 장애물에 부딪혀도 타이머가 초기화되지 않습니다.
